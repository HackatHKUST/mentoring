Git/Github
============
Version control for the rest of us
--------------------------

###What is Git?

Git is the tool that handles your version control, github is the online repository where you store all your files. You can find the simulation-version of this tutorial [here](try.github.io).

After installing the git command line tool and setting up my github account, I can do the following to start using git.

### To set up the local repository:
- CD into designated folder
- git init - that sets up the folder as a local repository
- git add (targets) - do this to make git track the files you 
- git add ‘*.txt’ adds anything that has a .txt ending
- git commit -m “message” to make commits to the local repository
